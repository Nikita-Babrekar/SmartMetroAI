# 🚆 SmartMetro AI

> AI-powered system for predicting metro crowd levels and optimizing travel decisions using smart recommendations and pricing strategies.

---

## 🌍 Problem

Urban metro systems face:

- 🚶‍♂️ Overcrowding during peak hours  
- 📉 Underutilization during off-peak hours  
- 💸 Fixed ticket pricing (no demand control)  
- ❌ No real-time crowd awareness  

👉 Result: Poor passenger experience & inefficient resource usage  

---

## 💡 Solution

**SmartMetro AI** uses Machine Learning to:

- 🔍 Predict crowd levels based on time & day  
- 📊 Provide real-time-like insights  
- 💰 Suggest smart pricing (incentive-based)  
- 🧭 Recommend optimal travel time  

---

## ⚙️ Features

- 🚦 Crowd Prediction (Low / Medium / High)  
- 💰 Smart Pricing Recommendation  
- 🧠 Travel Time Suggestions  
- 📊 Crowd Trend Visualization  
- 🌐 Interactive Web Dashboard  

---

## 🛠️ Tech Stack

- **Python**
- **Pandas, NumPy**
- **Scikit-learn**
- **Streamlit**
- **Joblib**

---

## 📊 How It Works

1. User selects:
   - Time  
   - Day (Weekday/Weekend)  

2. Model predicts crowd level  

3. System provides:
   - Crowd status  
   - Travel suggestion  
   - Pricing recommendation
